HANDWRITTEN-DIGIT RECOGNITION APP

Prediction: none

---------------------Project Description--------------------

  • This project enables the user to identify the handwritten digits of various kind through a model which is trained using mnist dataset which contains several handwritten digits.
  • This project is developed using various technologies like python,html,css,javascript.
  • Tensorflow is used as the machine learning platform for building this app.
  • Some basic knowledge of CNN and deep learning is required to understand the working of model.
  • The accuracy of this model is 98.81% on test data for now.
  • This project is created using google colab notebook and visual studio and then deployed to heroku.
